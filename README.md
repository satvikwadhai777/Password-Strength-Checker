# Password Strength Checker

This is a simple Python project that checks whether a password is strong or not.  
The user can enter a password, and the program analyzes it based on length, uppercase letters, lowercase letters, numbers, and special characters.

## Features
- Check password strength
- Shows whether the password is Weak, Medium, or Strong
- Gives improvement suggestions
- Checks password length
- Checks uppercase and lowercase letters
- Checks numbers and special characters
- Beginner-friendly Python project
